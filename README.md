# Astro+TinaCMS Portfolio
Portfolio is an Astro template built with of course Astro 5.7, React 19 and TailwindCSS 4

## Original Repo
Clone Repository
```sh
git clone https://github.com/iann-mathaiya/nikola-tesla.git
```

Install Dependencies
```sh
pnpm install
```

Development
```sh
pnpm run dev
```

Build
```sh
pnpm run build
```

Preview
```sh
pnpm run preview
```
